# Simple Interactive Web App

A clean, modern web application built with pure HTML, CSS, and JavaScript featuring interactive components and smooth animations.

## Features

### 🎯 Interactive Counter
- Increment, decrement, and reset functionality
- Color-coded display based on counter value
- Smooth animations and transitions
- Keyboard shortcuts (Ctrl/Cmd + +/- for increment/decrement, Ctrl/Cmd + R for reset)

### 🎨 Background Color Changer
- Five preset color options
- Smooth background transitions
- Ripple effect on button clicks
- Visual feedback with hover states

### ✅ Todo List
- Add, complete, and delete tasks
- Persistent storage using localStorage
- Click to mark tasks as completed
- Clean, intuitive interface

## Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Modern styling with animations and transitions
- **Vanilla JavaScript** - No frameworks, pure JS functionality
- **LocalStorage** - Client-side data persistence

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Start interacting with the features!

## Keyboard Shortcuts

- `Ctrl/Cmd + +` - Increment counter
- `Ctrl/Cmd + -` - Decrement counter  
- `Ctrl/Cmd + R` - Reset counter
- `Enter` - Add todo item (when input is focused)
